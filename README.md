# Mowika

A sleek, high-performance streaming discovery platform that pairs **The Movie Database (TMDB)** metadata with embedded video sources like **VidSrc** for a seamless viewing experience.

---

###  Features
* **Massive Library:** Access a **Discover** page with over **25,000+** movies and TV shows, including a dedicated section for **Anime**.
* **Dynamic Categories:** Stay updated with **Trending**, **Upcoming**, **Top Rated**, **Popular**, and **Now Playing** sections.
* **Personalization:** Manage your **Watchlist**, track your **History**, and explore deep-dive **People** bios and **Movie/Show Collections**.
* **Flexible Access:** Enjoy a **No-Login Required** experience for instant browsing, or use the **Auth System** to sync your data across devices.

### Technical Overview
The platform leverages the TMDB API to fetch real-time metadata and utilizes third-party embed providers to deliver content without hosting files directly. It is designed with a responsive, mobile-first interface for high-speed navigation.
